typeform-now
============
Deploy your typeform to a custom URL using now.sh

Installation
------------

```
$ npm install -g typeform-now
```

Usage
-----

### Deploy a typeform
```
$ typeform-now <your-typeform-url>
```

### Deploy a typeform locally
```
$ typeform-now --dev <your-typeform-url>
```

### Usage information
```
$ typeform-now --help
```
